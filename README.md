# vue-dayjs-sample

## 概要
dayjsの使い方練習用のサンプル集と、インスタンス生成時のプラグイン追加方法の共通化サンプル

### セットアップ  
```sh
npm install
```
  
### 実行  
```sh
npm run dev
```
